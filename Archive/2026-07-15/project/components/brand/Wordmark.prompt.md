The PLNLY / plainly lockup — the core brand mark; the wordmark never appears without "plainly" beneath it on first contact.

```jsx
<Wordmark variant="stacked" size="md" />
<Wordmark variant="horizontal" />          {/* headers, signatures */}
<Wordmark variant="compact" />              {/* the "P." disc — app, favicons */}
<Wordmark onInk />                          {/* reversed on a dark surface */}
<Wordmark subBrand />                       {/* PL·AI·NLY — the AI-literacy line */}
```

- `variant`: `stacked` (primary), `horizontal`, `compact` (P. disc).
- `reveal={false}` drops "plainly" — only for repeat contact or below minimum size (22px / 9mm).
- Never recolor the wordmark, stretch it, or set "plainly" upright. Clear space = cap-height of the "P".
